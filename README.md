# Healthcare-Data-Dashboard
_Description: Dynamic, interactive dashboards_
To elaborate, I analyzed an NJ public healthcare dataset to understand how neonatal health indicators have changed since 2000. By visualizing the data in Power BI, I was able to distinguish between 'predictable' periods of systemic stability and 'disruptive' periods of high variance. This project demonstrates my ability to clean large datasets and translate raw numbers into clinical/operational insights.
